# Period Tracker

这是一个可以添加到 iPhone 主屏幕的经期记录网页应用。

## 本地预览

当前电脑上已启动预览服务：

- Mac 本机打开：http://127.0.0.1:4174/
- iPhone 和 Mac 在同一 Wi-Fi 时打开：http://192.168.31.199:4174/

## 添加到 iPhone 主屏幕

1. 用 iPhone 的 Safari 打开上面的局域网地址。
2. 点底部分享按钮。
3. 选择“添加到主屏幕”。
4. 名称可保留“Period Tracker”，图标会使用附件图片。

## 隐私

记录保存在当前浏览器的本地存储里，不会上传到服务器。

## 使用

在日历上先点开始日，再点结束日，中间日期会自动填充为实际经期。默认经期为 6 天、周期为 33 天；记录多次后，会根据实际经期长度和开始日间隔自动更新平均值。

## 离线

部署到 HTTPS 后可使用离线缓存。本地局域网 HTTP 地址主要用于预览和添加主屏幕。
